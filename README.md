# Voice Of Thousands (천 개의 목소리)


#### 이 어플리케이션은 여러분의 목소리들을  사용하여 한 권의 책을 제작하는 프로젝트를 위한 것입니다. 여러분의 목소리를 들려주세요!
This application is for a audio book production project that uses your voice.Let's hear your voice!
